# 系统架构说明

## 概述

Breakup Recovery Squad 是一个基于多智能体协作的情感支持系统，采用 Streamlit 前端 + Agno 多智能体框架构建。

## 模块职责

### 1. 配置模块 (`config/`)
- **settings.py**: 环境变量加载、API 配置管理
- **constants.py**: 常量定义、枚举值

### 2. 智能体模块 (`agents/`)
- **base.py**: 智能体基类、模型初始化
- **therapist_agent.py**: 共情疗愈智能体
- **closure_agent.py**: 告别释怀智能体
- **routine_agent.py**: 恢复规划智能体
- **honesty_agent.py**: 清醒真话智能体
- **manager.py**: 统一调度管理器

### 3. 提示词模块 (`prompts/`)
- **system_prompts.py**: 系统指令模板
- **user_templates.py**: 用户输入组装模板

### 4. 工具模块 (`utils/`)
- **session_store.py**: 会话状态管理
- **validators.py**: 输入验证

### 5. UI 模块 (`ui/`)
- **sidebar.py**: 侧边栏配置
- **input_area.py**: 输入区域
- **chat_tabs.py**: 聊天窗口
- **common.py**: 通用组件

## 数据流

1. 用户在侧边栏配置 API Key 和模型参数
2. 用户在输入区域填写情感描述
3. 点击「一键生成」或在 Tab 中单独对话
4. Agent Manager 调度对应的 Agent 执行
5. Agent 返回结果，存储到 Session State
6. UI 渲染对话历史和新消息

## 扩展点

- 新增 Agent：继承 `BaseAgent`，在 `manager.py` 中注册
- 新增模型提供商：扩展 `create_model()` 函数
- 新增 UI 组件：在 `ui/` 目录下添加模块
