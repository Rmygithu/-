# 使用手册

## 快速开始

### 1. 启动应用

```bash
streamlit run app.py
```

### 2. 配置 API Key

1. 在侧边栏输入 MiMo API Key
2. 看到「API Key 已配置！✅」提示即可

### 3. 使用方式

#### 方式一：一键生成全套方案

1. 在「Share Your Feelings」区域描述你的感受
2. 点击「Get Recovery Plan 💝」按钮
3. 等待 4 个 Agent 分析完成
4. 切换 Tab 查看各 Agent 的分析结果

#### 方式二：单独对话

1. 切换到对应的 Tab（情绪疗愈/告别释怀/恢复规划/清醒真话）
2. 在底部输入框输入消息
3. 按回车或点击发送
4. 与 Agent 进行多轮对话

## 功能详解

### 🤗 情绪疗愈 (Therapist Agent)

- 提供情感支持和倾听
- 验证你的感受
- 给予温暖的鼓励

**适用场景：**
- 感到悲伤、失落时
- 需要有人倾听时
- 寻求情感共鸣时

### ✍️ 告别释怀 (Closure Agent)

- 帮助撰写未发送的消息
- 引导情感释放
- 提供放下策略

**适用场景：**
- 想表达但未说出口的话
- 需要 closure 时
- 学会放下时

### 📅 恢复规划 (Routine Agent)

- 设计 7 天恢复计划
- 提供活动建议
- 规划日常作息

**适用场景：**
- 需要重建生活节奏时
- 想要充实自己时
- 寻找恢复动力时

### 💪 清醒真话 (Honesty Agent)

- 提供客观分析
- 指出成长空间
- 给出理性建议

**适用场景：**
- 需要理性看待关系时
- 想了解问题根源时
- 寻找前进方向时

## 对话历史

### 查看历史
- 每个 Tab 独立保存对话历史
- 滚动查看历史消息
- 区分用户消息和 Agent 回复

### 清空历史
- 点击侧边栏「🗑️ 清空全部对话」按钮
- 所有 Tab 的对话历史将被清空

## 高级配置

### MiMo 模型配置

在侧边栏展开「⚙️ 高级配置」：

- **API Base URL**: MiMo API 端点地址
- **Model ID**: 模型标识符

## 常见问题

### Q: 为什么 Agent 没有响应？
A: 请检查：
- API Key 是否正确
- 网络连接是否正常
- API 额度是否充足

### Q: 如何获得更好的回复？
A: 建议：
- 详细描述你的感受
- 尝试与多个 Agent 对话

### Q: 对话历史会保存吗？
A: 对话历史保存在内存中，刷新页面会丢失。建议重要内容及时复制保存。

### Q: 可以同时使用多个 Agent 吗？
A: 可以。使用「一键生成」功能会依次调用所有 Agent，也可以在不同 Tab 中分别对话。

## 隙私说明

- 对话历史不会上传到服务器
- API Key 仅用于调用模型服务
- 请勿输入敏感个人信息
